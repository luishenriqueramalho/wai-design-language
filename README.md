# WAI Design Language

Repositório oficial da linguagem visual, verbal e de produto da WAI.

A WAI Design Language existe para manter consistência entre marca, produto, landing pages, redes sociais e interfaces do ecossistema WAI, começando pelo WAI Risk.

## Objetivo

Criar uma base única para decisões de design, comunicação e experiência de usuário.

Este repositório não é apenas um manual visual. Ele define como a WAI deve parecer, falar, organizar interfaces e transmitir confiança em produtos digitais com inteligência artificial aplicada a decisões de risco.

## Princípios

- Clareza antes de estética.
- Confiança antes de impacto visual gratuito.
- Tecnologia com percepção premium, mas sem exagero futurista.
- Produto sério, objetivo e orientado a decisão.
- Linguagem simples, profissional e segura.

Primeiro produto do ecossistema WAI. Plataforma para análise visual de risco automotivo com apoio de inteligência artificial.

## Regra central

A WAI deve comunicar inteligência, segurança e precisão.

Não deve parecer genérica, amadora, infantil ou excessivamente decorativa.
