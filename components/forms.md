# Forms

Formulários da WAI devem ser claros, previsíveis e objetivos.

## Regras

- Label sempre visível.
- Placeholder não substitui label.
- Erro deve explicar o problema e como corrigir.
- Campos obrigatórios precisam estar claros.
- Não pedir dado que o fluxo não usa.

## Estados

- Default
- Focus
- Filled
- Disabled
- Error
- Success
- Loading

## Mensagens de erro

Bom:

> Informe a placa do veículo.

> A quilometragem deve ser maior que zero.

Ruim:

> Campo inválido.

> Erro.

## Regra prática

Formulário bom reduz dúvida. Formulário ruim empurra suporte para operação.
