# Botões

Botões devem deixar claro o que acontece após o clique.

## Tipos

| Tipo | Uso |
|---|---|
| Primary | Ação principal da tela |
| Secondary | Ação alternativa importante |
| Ghost | Ação discreta |
| Danger | Ação destrutiva ou crítica |

## Regras

- Uma tela deve ter uma ação primária dominante.
- Texto do botão deve começar com verbo quando possível.
- Evitar botões genéricos como “Ok” quando a ação for importante.
- Ação destrutiva precisa ser visualmente clara.

## Exemplos

Bom:

- Criar vistoria
- Gerar análise
- Salvar alterações
- Cancelar assinatura

Ruim:

- Prosseguir
- Confirmar
- Clique aqui
- Ok
