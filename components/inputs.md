# Inputs

Inputs are the main communication point between user and system.

## Types

- Text
- Search
- Password
- Number
- Currency
- Date
- Phone
- Plate
- VIN
- Textarea

## Sizes

- Small: 40px
- Medium: 48px
- Large: 56px

## States

- Default
- Hover
- Focus
- Filled
- Error
- Disabled
- Read only
- Loading

## Rules

- Label stays above the field.
- Do not use floating labels.
- Use 16px radius.
- Use 16px horizontal padding.
- Use clear helper and error text.
