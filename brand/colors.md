# Cores

As cores da WAI devem transmitir inteligência, segurança, tecnologia e precisão.

A paleta deve evitar aparência infantil, chamativa demais ou genérica de startup.

## Direção visual

- Base escura premium.
- Contraste alto para leitura.
- Azul como cor principal de confiança e tecnologia.
- Ciano como acento de inteligência artificial e precisão.
- Neutros frios para interface e documentação.

## Paleta conceitual

| Papel | Uso |
|---|---|
| Primary | Ações principais, destaques, links importantes |
| Accent | IA, insights, detalhes visuais e estados especiais |
| Background | Superfícies principais em dark mode |
| Surface | Cards, painéis, modais e containers |
| Border | Separadores e estrutura visual |
| Text Primary | Texto principal |
| Text Secondary | Texto auxiliar |
| Success | Estados positivos e conclusão |
| Warning | Atenção e risco moderado |
| Danger | Erro, bloqueio ou risco crítico |

## Regra prática

A cor não deve carregar significado sozinha. Sempre combine cor com texto, ícone ou estrutura visual.

## Evitar

- Gradientes excessivos.
- Cores neon sem função.
- Paleta colorida demais.
- Vermelho como elemento decorativo.
- Baixo contraste em textos críticos.
