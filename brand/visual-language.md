# Visual Language

A WAI deve parecer clara, precisa, silenciosa e confiável.

## Direção

- Quiet Intelligence.
- Precision Minimalism.
- Poucas cores.
- Muito espaço negativo.
- Tipografia protagonista.
- Interface calma.
- Baixo ruído visual.

## O que absorvemos

- Apple: silêncio visual.
- Linear: consistência de produto.
- Stripe: percepção de valor.
- Mercury: conforto visual.
- Aesop: elegância editorial.

## O que recusamos

- Neon.
- Cyberpunk.
- Glow exagerado.
- Glassmorphism como linguagem principal.
- Interface poluída.
- Dashboard com excesso de cards.

## Regra central

A interface nunca compete com a informação.
