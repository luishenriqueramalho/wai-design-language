# AI Usage Rules

Este arquivo orienta como uma IA deve interpretar e aplicar a WAI Design Language.

## Regras principais

- Priorize clareza antes de estética.
- Use poucos elementos visuais.
- Não invente cores fora dos tokens.
- Não use animações chamativas.
- Não use linguagem exagerada.
- Não prometa certeza absoluta.
- Sempre favoreça evidências, contexto e decisão humana.

## Ao gerar interfaces

A IA deve preservar:

- hierarquia visual;
- espaçamento generoso;
- baixo ruído visual;
- componentes simples;
- linguagem objetiva;
- foco na decisão do usuário.
