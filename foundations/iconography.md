# Iconography

The official icon library is Lucide.

## Sizes

- 16px for inline text and compact UI.
- 20px for inputs and small actions.
- 24px for buttons and navigation.
- 32px for empty states and large UI blocks.

## Stroke

Use 2px stroke as default.

## Rules

- Icons must support meaning, not decorate.
- Do not mix filled and outline icons.
- Do not color icons without functional meaning.
- Icon color should follow text color unless the icon indicates status.
