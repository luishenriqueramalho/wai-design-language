# Espaçamento

O espaçamento da WAI deve criar interfaces organizadas, respiráveis e fáceis de escanear.

## Direção

- Layouts com respiro.
- Densidade equilibrada para dashboards.
- Agrupamento visual claro.
- Evitar telas apertadas ou poluídas.

## Escala conceitual

| Token | Uso |
|---|---|
| xs | Espaços internos pequenos |
| sm | Relação próxima entre elementos |
| md | Espaçamento padrão entre blocos |
| lg | Separação entre seções |
| xl | Áreas de destaque |
| 2xl | Heros e composições institucionais |

## Regra prática

Elementos relacionados ficam próximos.

Blocos com responsabilidades diferentes precisam de separação clara.
