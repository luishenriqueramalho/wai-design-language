# Dashboard Pattern

Na WAI, dashboard deve funcionar como centro de decisão.

## Perguntas que a tela deve responder

1. Existe algo urgente?
2. Como está a operação?
3. O que mudou?
4. O que precisa de atenção?
5. Qual decisão deve ser tomada?

## Regras

- Evitar excesso de cards.
- Evitar gráficos sem ação clara.
- Priorizar narrativa de decisão.
- Mostrar primeiro o que importa.
- Mostrar depois o que explica.
- Mostrar por último o que complementa.
